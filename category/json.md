javascript에서 데이터를 주고 받기 위한 방식으로 사용되는 **javascript object notation (json)** 에 대해 학습합니다.


```javascript
var json =
{
  "first" : "this",
  "second" : "is",
  "third" : "json"
};

console.log(json.first +" "+ json.second +" "+ json.third);
```


```
this is json
```

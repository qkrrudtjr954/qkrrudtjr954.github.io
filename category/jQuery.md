javascript의 가장 대중적인 라이브러리인 ***jQuery*** 를 배워봅니다.


```html
<!DOCTYPE html>
<html>
  <head>
    <meta charset="utf-8">
    <title>jQuery!</title>
  </head>
  <body>
    <div id="jQuery">
      <p></p>
    </div>

    <script type="text/javascript">
      $('div#jQuery p').text('jQuery를 배워봅시다.');
    </script>
  </body>
</html>
```

ajax 는 jQuery 소속으로 데이터를 페이지 전화없이 얻어올수 있는 비동기식 처리를 하는 기술을 의미합니다. <br>
jQuery는 어떤 기능을 담당하는지 어떻게 활용하는지 학습합니다.


```js
$(function(){
  $.ajax({
    type:'post',
    data: {
      data1 : 'this',
      data2 : 'is',
      data3 : 'Ajax!'
    }
  })
})
```
